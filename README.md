# Set-TLS
### Synopsis
Configure a device to use TLS 1.2 for secure communications

### Description
* Configures the .NET Frameworks to use Strong Crypto
* Disables the SSL 2.0/3.0 and the TLS 1.0/1.1 Protocols 
* Enables the TLS 1.2 Protocol
